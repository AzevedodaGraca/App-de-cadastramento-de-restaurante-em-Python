# 🍽️ Restaurant Registration App in Python

## 📋 Description

A console application developed in Python to manage a restaurant registration system. The system allows you to register new restaurants, view a list of registered restaurants, toggle activation/deactivation status, and manage categories.

## ✨ Features

- **Register Restaurant**: Add new restaurants with name and category
- **List Restaurants**: View all registered restaurants with their status
- **Toggle Status**: Activate or deactivate restaurants as needed
- **Intuitive Interface**: Main menu with clear and easy-to-use options
- **Status Management**: Control which restaurants are active or inactive

## 🛠️ Technologies Used

- **Python 3.x**
- **Standard Libraries**: `os`

## 📦 Prerequisites

- Python 3.6 or higher installed

## 🚀 How to Use

1. **Clone the repository**:
   ```bash
   git clone https://github.com/AzevedodaGraca/App-de-cadastramento-de-restaurante-em-Python.git
   cd App-de-cadastramento-de-restaurante-em-Python
   ```

2. **Run the application**:
   ```bash
   python app.py
   ```

3. **Interact with the menu**:
   - Type `1` to register a new restaurant
   - Type `2` to list all restaurants
   - Type `3` to toggle the status of a restaurant
   - Type `4` to exit the application
